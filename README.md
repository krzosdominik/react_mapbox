<!-- GETTING STARTED -->
## Getting Started

### General info
ReactMapbox App. Technologies: React, MapBox GL and Now UI Kit

### Prerequisites

* Node js
```sh
https://nodejs.org/dist/v12.15.0/node-v12.15.0-x64.msi
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/krzosdominik/react_mapbox.git
```
2. Install NPM packages
```sh
npm install
```
3. Run app
```sh
npm start
```
